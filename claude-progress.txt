### Priority 1 — Demo Page (VAPI-001 to VAPI-020)
The current app/page.tsx is a static marketing page. It needs:
- A VapiCallButton component with these states:
  * idle: blue bg, microphone icon, "Start Call" accessible label
  * connecting: "Connecting..." text, disabled state
  * active: "End Call" red bg, visible call timer (MM:SS)
  * ended: resets to idle, timer to 00:00
- Live transcript area (aria-live="polite") that updates from vapi.on('message')
- Speaking indicator (shows on 'speech-start', hides on 'speech-end')
- Outbound phone number form with:
  * E.164 validation (+15551234567 valid, 'not-a-phone' invalid)
  * Auto-format display (5551234567 → (555) 123-4567)
  * POST to /api/calls/outbound on submit
- ErrorBoundary wrapping the call UI section
- Vapi SDK init: vapi.start(assistantId) on call start, vapi.stop() on end call
